
# AI CV Builder Pro

Premium AI-powered global CV builder.

## Features
- AI resume generation
- Country-specific CV styles
- Mobile-first responsive design
- Authentication ready
- Database ready
- PDF export ready
- Modern UI
- Dashboard

## Setup

```bash
npm install
npm run dev
```

## Required Environment Variables

```env
OPENAI_API_KEY=
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
CLERK_SECRET_KEY=
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
```
